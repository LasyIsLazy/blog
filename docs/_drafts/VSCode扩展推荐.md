添加书签

## ESLint

集成 ESLint

## Excel Viewer

以表格的形式查看 VSC 和 Excel 文件

Go to definition

将文件夹生成字符形式的文档树

查看 Git 历史记录

## Node.js Modules Intellisense

在浏览器中打开当前文件

代码格式化

## Project Manager

项目管理器：快速在不同项目之间切换

## stylelint

CSS/SCSS/Less lint

以树状结构显示项目中所有的 TODO、FIXME 等

## TSLint

tslint

## Vetur

Vue 支持
